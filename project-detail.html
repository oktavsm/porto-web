Aplikasi Mobile "MUEL"

Aplikasi cerdas untuk membantu pengguna melakukan monitoring emosi dan memandu gaya hidup sehat secara personal.

Latar Belakang Proyek

Kesehatan mental dan emosional adalah kunci kesejahteraan sehari-hari. Proyek "MUEL" lahir dari kebutuhan antarmuka interaktif yang dapat memetakan perasaan harian seseorang sekaligus menyarankan perubahan kecil pada rutinitas mereka.

Fokus utama dari aplikasi ini adalah memberikan user experience yang bersih, menyenangkan (friendly), namun juga memiliki backend logic yang kokoh untuk memproses dan menyimpan log emosi pengguna dengan aman.

Teknologi yang Digunakan:

  • Bahasa: Kotlin
  • Database: Firebase Realtime DB / Firestore
  • Infrastruktur: Google Cloud Console (GCP)
  • IDE: Android Studio

Fitur Utama

Perekam Emosi

Log harian yang interaktif untuk mencatat keadaan mood dan faktor penyebabnya.

Dasbor Analitik

Visualisasi data dalam bentuk grafik, melihat pola perasaan dari waktu ke waktu.

Sinkronisasi Cloud

Integrasi cerdas mem-backup data ke Firebase & layanan Google Cloud untuk akses multi-perangkat.
