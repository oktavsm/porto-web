Tentang Saya

Sebagai mahasiswa Teknik Informatika Universitas Brawijaya dengan IPK hampir sempurna (3.98/4.00) sekaligus penerima Beasiswa Kepemimpinan TELADAN, saya memiliki antusiasme tinggi terhadap Rekayasa Perangkat Lunak dan Kecerdasan Buatan (AI). Saat ini, saya aktif sebagai Anggota Laboratorium Information Centric Networking (ICN), di mana saya memperdalam minat saya pada infrastruktur jaringan khususnya di ranah cloud computing dan sistem terdistribusi.

Di samping itu, saya juga terus mengasah pengalaman praktis dalam pengembangan Aplikasi Mobile menggunakan Kotlin dan Android Studio. Berbekal fondasi pemrograman dan logika algoritmik yang kuat, saya juga dipercaya menjadi Asisten Praktikum Pemrograman Dasar (Java), Sistem Operasi dam Pengantar Jaringan Komputer, di mana saya aktif melatih skill mengajar dan juga berbagi pengetahuan bersama rekan-rekan mahasiswa lainnya.
